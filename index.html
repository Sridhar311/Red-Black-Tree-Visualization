What is a Red-Black Tree?

A Red-Black Tree is a type of binary search tree that maintains balance through specific properties:

  • Every node is either red or black.
  • The root is always black.
  • Red nodes cannot have red children (no two red nodes can be adjacent).
  • Every path from a node to its descendant NULL nodes has the same number of black nodes.

This project provides an interactive visualization of Red-Black Trees, allowing you to insert, delete, and explore nodes.
